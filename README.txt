В программе 2 основных класса - atom2D и connection, описание которых находится в одноименных файлах cpp и h.
Главная функция моделирования Modeling() находится в файле modeling.*
Моделирование делится на два этапа: 
1) Создание начальной конфигурации. Функция start_new_system1().
2) Интегрирование уравнений движения, с помощью функции Step1().

Главными данными в программе являются 2 vector-a.
Particles (от класса atom2D) содержит в себе все частицы, используемые при моделировании
connect_list (от класса connection) хранит все связи, используемые при моделировании

Начальная конфигурация создается по следующиму принципу:
1) В область "накидываются" частицы. Каждая частица проверяется, может ли она там находится.
2) Когда необходимое число частиц набросано, частицы соединяются друг с другом связами.

Интегрирвоание уравнений движения происходит по алгоритму
1) Проход по всем частицам. Определение результирующей силы, действующую на частицу проходом по связям имеющимся у частицы.
2) Определение скоростей и перемещений всех частиц.
